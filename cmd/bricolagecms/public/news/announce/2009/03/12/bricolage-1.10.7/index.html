Bricolage 1.10.7 Released

2009.03.12

The Bricolage development team is pleased to announce the release of Bricolage 1.10.7, the latest in the stable 1.10 line of Bricolage. This maintenance release includes important new features and improvements that will improve your Bricolage experience both behind the scenes and in direct use. Some of the highlights include:

Stability

The new Bricolage is vastly more stable in triggered publishing, media management, story versioning, and template management. It looks better, too!

Gearing up

With the long-awaited release of Bricolage 2.0 just around the corner, most new features are now being built for the upcoming 2.0 release candidate. This 1.10 series release focuses on stability and bug fixes.

More Under-the-hood Improvements

There are many more changes in Bricolage 1.10.7; For a complete list, see the changelog. For the complete history of ongoing changes in Bricolage, see Bric::Changes

Download Bricolage 1.10.7 now from the Bricolage Web site Downloads page, from the SourceForge download page, and from the Kineticode download page.

About Bricolage

Bricolage is a full-featured, enterprise-class content management and publishing system. It offers a browser-based interface for ease-of use, a full-fledged templating system with complete HTML::Mason, HTML::Template, PHP5, and Template Toolkit support for flexibility, and many other features. It operates in an Apache/mod_perl environment and uses the PostgreSQL for its repository. A comprehensive, actively-developed open source CMS, Bricolage has been hailed by eWEEK as quite possibly the most capable enterprise-class open-source application available.
