Instructions

Select class names from the navigation tree to the left. The tree shows a hierarchical list of modules and programs. In addition to this URL, you can link directly to the page for a particular module or program. For example, if you wanted to access Bric::Admin, any of these links will work:

  • /?Bric::Admin
  • /Bric::Admin
