Public Instance Methods

$self = $coll->save($obj, $map_class)

Saves the changes made to all the objects in the collection. $obj contains the objec to which new contacts should be mapped, and $map_class holds the name of the Bric::Util::Map subclass that does the mapping.

Throws:

  • Bric::_get() - Problems retrieving fields.

  • Incorrect number of args to Bric::_set().

  • Bric::set() - Problems setting fields.

  • Unable to connect to database.

  • Unable to prepare SQL statement.

  • Unable to execute SQL statement.

  • Unable to select row.

Side Effects: Uses a subclass of Bric::Util::Map internally.
