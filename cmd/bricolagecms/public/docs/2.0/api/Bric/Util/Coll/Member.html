Public Instance Methods

$self = $coll->del_mem_obj($obj, $mem)

Deletes a member from the collection. This method should only be called internally by Bric::Util::Grp when it detects that the group object contains members that are only one particular class of object. That is, for groups that cannot contain different classes of objects. The collections are build by the Bric::Util::Grp::Parts::Member class' href() method, which uses the object IDs for the keys when only one type of object is managed, and the member ID otherwise. We have to add this special method in order to manage those situations in which only one type of object is managed, so that the member to be deleted from the collection is stored under the ID of its associated object, rather than its own ID. This allows get_objs() to do the right thing and exclude deleted objects, while at the same time allowing save() to get access to the actual member objects that need deleting.
