DESCRIPTION

This document points to the various documents written to document the process of implementing the element revision in Bricolage. The functional specification describes the new element system, what it looks like, and how it works. The technial specification is an implementation plan that describes how Bricolage will be changed to implement the element revision. The implementation document documents everything done to actually implement the new element system, including how the final product differs from that described in the functional and technical specifications.
