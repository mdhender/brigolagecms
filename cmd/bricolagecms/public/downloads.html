How to get Bricolage

Bricolage is free, open source software released under the BSD license. It requires Perl version 5.8.0 or higher (5.8.8 is the minimum recommended), and it has dependencies on a several Perl libraries, as well as Apache, mod_perl, PostgreSQL or MySQL, and Expat (which you should probably install and configure before proceeding).

There are two ways you can get Bricolage:

Option 1. Download the latest stable version

The latest stable version is 2.0.1 and you can download it directly here.

Option 2. Get the latest version from Github

The latest Bricolage version is the one that's in our GitHub repository. Get it using this shell command, which requires Git:

git clone git://github.com/bricoleurs/bricolage.git

Alternately, you can use the "Download" link on the Github page to download a .zip or .tar.gz archive.

Just evaluating? Try the VMware or QEMU image

Adams University has been kind enough to make up-to-date versions of Bricolage available as both VMware and QEMU images. You can download the images from their public FTP site here.

After you get it

See the administration guide for installation instructions.

And be sure to sign up for the bricolage-users mailing list, where other Bricolage users and the Bricolage developers themselves all hang out to help each other.
