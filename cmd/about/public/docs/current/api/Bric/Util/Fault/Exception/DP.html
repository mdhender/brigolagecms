Name

Bric::Util::Fault::Exception::DP - Legacy Exception Class

Description

This is a dummy class to keep Bricolage from breaking when various classes and libraries use Bric::Util::Fault::Exception::DP. Eventually all such use statements will be removed and we'll be able to delete this file, as all exceptions are now handled internally by Bric::Util::Fault.
