Name

Bric::Util::ApacheConst - Common Apache and HTTP status code constants

Synopsis

use Bric::Util::ApacheConst qw(:common);
use Bric::Util::ApacheConst;

Description

This package implements common constants used by mod_perl handlers for return values and HTTP status codes so that Bricolage doesn't have to care about which version of Apache is running.

Author

Scott Lanning <slanning@cpan.org>
